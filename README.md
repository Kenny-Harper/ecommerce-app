## Intro
A fully functional e-commerce product page built with React and Vite.

## Features
- Product grid with category filtering
- Add to cart with quantity controls
- Sliding cart sidebar
- Full checkout form with validation
- Order confirmation screen
- Cart persists with localStorage

## Tech Stack
- React
- Vite
- CSS (custom styling)

## Concepts Used
- useState for cart, filter and checkout state
- Props and component structure
- Controlled form inputs with validation
- Conditional rendering
- Array methods (.map, .filter, .find)
- localStorage persistence
